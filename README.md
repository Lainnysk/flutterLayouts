# FireStore & FireStorage
## Работа с базой данных FireStore и хранилищем FireStorage

__Цель работы:__

Освоить навыки работы с FireStore и FireStorage, реализовать сохранение данных при регистрации пользователя в firestore, редактирование данных пользователя в firestore, работу с данными CRUD, динамическое обновление данных, функционал для загрузки и удаления картинок в Firebase Storage, регистрацию и авторизацию, отображение списка всех картинок, находящихся в Firebase Storage, харанение образа изображения в облаке store и само изображение в хранилище storage.
